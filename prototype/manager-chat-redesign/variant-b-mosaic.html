Der Manager kann jedes Tile "spotlighten" — dann wird es zum großen Tile hochgezogen.

Wie es sich anfühlt

  • Drei klare Zonen übereinander: Chat-Tile, Spotlight-Tile, Mini-Tile-Grid. Jede Zone macht eine Sache richtig gut.
  • Der Spotlight-Bereich ist die "Bühne" — hier landet automatisch das Terminal, das gerade Aufmerksamkeit braucht (z.B. Test-Fehler). Manuell pinnen via Long-Press auf eine Tile.
  • Das 2-Spalten Grid zeigt 6 Terminals gleichzeitig sichtbar, weitere durch Scroll. Jede Tile hat eine Sparkline die Output-Aktivität über Zeit zeigt — du siehst auf einen Blick wo gerade was passiert.
  • Die @mention im Chat markiert die zugehörige Tile mit Glow — wieder probier es, klick auf eine Pille.
  • Filter-Chips ("Alle / Aktiv / Fehler") über dem Grid für schnelles Eingrenzen wenn 10 Terminals zu viel werden.
  • Kompakter als A — du siehst den Chat, das wichtigste Terminal UND alle anderen in EINEM Screen ohne Scrollen.
