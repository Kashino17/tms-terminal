Was hier alles funktioniert

  • Tool-Sidebar (links): 6 Tools verfügbar — 🔧 Werkzeuge, ⚡ Quick Actions, 📋 Snippets, 🗂 Files, 🔍 Search, 🤖 AI. Plus + für eigene Tools.
  • Sidebar Auf-/Zuklappen: Klick den Chevron oben links — Sidebar wird breiter (zeigt Labels) oder schmal (nur Icons).
  • Tool-Flyout: Klick auf ein Tool öffnet ein Panel rechts daneben. Werkzeuge zeigt Aktionen für aktives Pane (Stop, Restart, Clear, Kill). Quick zeigt häufige Befehle (git status, npm install). Snippets zeigt eigene Befehl-Templates.
  • Quick-Actions klicken pusht den Befehl direkt ins aktive Pane mit grüner Flash-Animation am Pane-Rand.

Direct-Terminal-Mode (NEU im Input)

  • Mode-Toggle ganz links im Input: 💬 Chat / ▶ Terminal
  • Chat-Mode (default): Nachricht geht an den Manager (wie bisher)
  • Terminal-Mode: Klick auf ▶ → Input wechselt zu mono-Font, dunklerem Background, grünem Send-Button + Prefix @test-runner ▶
  • Tippe einen Befehl + Enter → Befehl wird direkt ins aktive Pane geschrieben mit Flash-Animation
  • Klick Pane = aktives Terminal wechseln. Prefix updates automatisch.
