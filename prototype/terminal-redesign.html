Design-Unterschiede

  • Clean Minimal: Header zeigt Server-Info, Tap darauf öffnet Quick Actions Drawer. Tabs bleiben als einzelne Zeile. Tools über einen Floating Action Button (rechts unten) erreichbar. Alle Features da, aber hinter einem Tap versteckt.
  • Floating Controls: Noch minimaler — Header hat nur Name + ⋯ Menü. Tabs sind halbtransparente Pills die über dem Terminal schweben. Tool-Buttons sind glasartige Floating-Icons am rechten Rand. Alles über dem Content, kein Platz verschwendet.
  • Gesture-First: Maximum Terminal-Fläche. Keine Tab-Bar — Dots unten + klickbare Ränder. Rechtsklick öffnet Tool-Menü. Pfeiltasten ← → wechseln Tabs. G = Tab Grid, T = Tools, ? = Hilfe. Klick auf den Titel = Tab Grid.
