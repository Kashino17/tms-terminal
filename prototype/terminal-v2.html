Alle 3 basieren auf "Floating Controls" — hier sind die Unterschiede:

  • Glass Layer: Header + Tabs + Tool-Buttons sind separate Glasflächen die über dem Terminal schweben. ⋯ öffnet ein Bottom-Sheet mit allen Features. Klar getrennte Schichten — du siehst wo alles ist.
  • Immersive: Zero UI by default — nur Terminal. Header + Toolbar erscheinen beim Hovern über den oberen/unteren Rand. Alles andere via ⌘K Command Palette (wie VS Code/Spotlight). Maximale Immersion, alles durchsuchbar.
  • Dynamic Island: Ein einzelner Pill oben vereint Tabs + Tools + Status. Tap expandiert, zweiter Tap klappt zu. Unten ein Mini-Pill mit den wichtigsten Keys. Alles in 2 Elementen — kein Header, keine Sidebar, keine Tab-Bar.
